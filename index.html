CP demos

We collect a few web based demonstrator applications in this website.

These are proof of concept projects to showcase how our pipeline of software tools work.

They all use Conjure to solve the respective problems, but the user interface is developed by project students at the School of Computer Science at the University of St Andrews.

People involved in developing and maintaining each demo are referenced in the relevant pages.

The source code for each demo is hosted in a Github repository. Please feel free to let us know if there are any issues with the demos using the issue tracker in the relevant repository.
